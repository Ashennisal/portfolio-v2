# Ashen Nisal - Personal Portfolio

A modern, dynamic personal portfolio website built with **Next.js**, **React**, and **Tailwind CSS**. Designed to showcase my skills as a Software Engineer and AI Specialization Student at SLIIT.

## Features

- **Modern UI/UX**: Sleek, dark-themed design with glassmorphism effects and smooth backdrop blurs.
- **Responsive Grid Layout**: Adaptive, bento-box style grid that looks great on both desktop and mobile devices.
- **Dynamic Content Tiles**:
  - Identity & Introduction
  - Skills and "Neural Stack"
  - Highlighted AI Projects (e.g., Legal Analyzer)
  - Recent Software Engineering Labs (Java, Spring Boot, Next.js)
  - Live local time and availability status
  - Downloadable Curriculum Vitae
- **Animations**: Subtle pulse and morphing animations for an engaging user experience.

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- [React](https://react.dev/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- TypeScript

## Deployment

This portfolio is optimized and deployed on **Vercel**, the native hosting platform for Next.js applications, ensuring fast global delivery and seamless updates.

Open [https://portfolio-v2-chi-rosy.vercel.app/](https://portfolio-v2-chi-rosy.vercel.app/) with your browser to see the live result.
